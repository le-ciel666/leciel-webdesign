株式会社D-ONE — ALC工事 コーポレートサイト
==========================================

■ ファイル構成
  index.html       … メインページ（このファイルを開くとサイトが表示されます）
  styles.css       … スタイルシート
  app.js           … ナビ・スクロール演出・フォーム・カラー切替
  image-slot.js    … 写真プレースホルダー用スクリプト

■ デプロイ方法（静的サイト）
  この4ファイルを同じ階層のまま、サーバーの公開フォルダ
  （例: /public_html/ など）にアップロードするだけで公開できます。
  Netlify / Vercel / Cloudflare Pages などへフォルダごとドラッグ＆
  ドロップでも公開可能です。トップに index.html があるため、
  ドメイン直下で自動的に表示されます。

■ 写真について（重要）
  現在、現場・スタッフ写真はプレースホルダー（差し替え枠）です。
  本番公開時は、各枠に実際の写真を入れてください。
  ・ご希望があれば、お預かりした写真を直接埋め込んだ
    「本番用ファイル」をご用意します（推奨）。

■ 仮の情報
  電話番号（044-XXX-XXXX）・住所・設立年などは仮です。
  index.html 内の該当箇所を正式な情報に置き換えてください。

■ お問い合わせフォーム
  デモ用（送信時は完了メッセージのみ表示）です。
  実際のメール送信には、サーバー側の処理（PHP / フォーム
  サービス / WordPress プラグイン等）の組み込みが必要です。
