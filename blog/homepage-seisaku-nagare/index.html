ホームページ制作の流れ｜依頼から公開までの全ステップ・期間・準備するもの

公開日：2026年6月20日 カテゴリ：基礎知識

「ホームページを作りたいけれど、依頼したら何をどう進めるのか分からない」——はじめての方ほど、この不安で一歩を踏み出せずにいます。 結論から言うと、制作の流れは「相談 → ヒアリング → 設計 → デザイン → 実装 → 公開 → 運用」のステップで進み、難しい専門作業はすべて制作側が担当します。 あなたがやることは、目的を伝え、内容を確認し、写真や文章を用意すること。この記事では、各ステップで「何が起き、自分は何をすればいいのか」と、期間の目安・準備するもの・スムーズに進めるコツまで、依頼前の不安が消えるように解説します。

ホームページ制作の流れ【全体早見表】

まずは全体像をつかみましょう。依頼から公開までは、おおむね次の流れで進みます。「誰がやるか」に注目すると、自分の役割が見えてきます。

ステップ主な内容誰がやる
1. 問い合わせ・相談やりたいこと・予算感を伝える施主
2. ヒアリング・目的整理目的・ターゲット・必要ページを整理一緒に
3. 見積もり・契約金額・範囲・納期を確認して合意一緒に
4. 構成・設計（ワイヤーフレーム）ページ構成と情報の並べ方を設計制作側
5. デザイン制作見た目を作り、確認してもらう制作側 → 施主確認
6. 原稿・写真の準備文章と写真を用意する（4〜6と並行）施主（支援可）
7. コーディング・実装Webサイトとして組み立てる制作側
8. 確認・修正表示や内容をチェックして直す一緒に
9. 公開・納品サーバーに公開して引き渡す制作側
10. 運用・保守更新・改善・管理を続ける一緒に

専門作業はすべて任せられます

サーバー・ドメインの設定、デザイン、コーディングといった専門的な部分は、すべて制作側が担当します。施主側のメインの役割は、「目的を伝える」「内容を確認する」「写真や文章を用意する」の3つ。難しい知識は必要ありません。

依頼から公開までの全ステップ詳細

各ステップで実際に何が起きるのかを、もう少し詳しく見ていきましょう。

STEP1. 問い合わせ・相談

問い合わせフォームやメールで、「どんなサイトを作りたいか」「いつ頃・いくらくらいで考えているか」を伝えます。 この段階では、内容が固まっていなくてまったく問題ありません。「集客したい」「古いので新しくしたい」程度のざっくりした希望でOKです。多くの制作者は初回相談・見積もりを無料にしています。

STEP2. ヒアリング・目的の整理

打ち合わせ（オンライン可）で、サイトの目的・ターゲット・伝えたい強み・必要なページなどを一緒に整理します。ここがサイトの成果を最も左右する重要工程です。 「誰に・何を伝えて・どう行動してほしいか」がはっきりするほど、成果につながるサイトになります。個人や小規模事業でホームページが必要かどうか迷っている方は個人事業主にホームページは必要？もご覧ください。

STEP3. 見積もり・契約

ヒアリング内容をもとに、金額・作業範囲・納期・修正回数などをまとめた見積もりが出ます。内容に納得したら契約に進みます。 「どこまでが料金に含まれるか」「公開後の更新は別料金か」を必ず確認しましょう。費用の考え方はホームページ制作の費用相場、依頼先の選び方はフリーランスと制作会社の違いが参考になります。補助金を使える場合もあります（参考：制作に使える補助金・助成金2026）。

STEP4. 構成・ワイヤーフレーム（設計）

色やデザインの前に、「各ページに何を・どの順番で並べるか」という設計図（ワイヤーフレーム）を作ります。 情報の順番は、読みやすさと問い合わせ率を大きく左右します。見た目より先に「中身の骨組み」を決めるのがプロの進め方です。

STEP5. デザイン制作

骨組みに沿って、事業の世界観に合った見た目を作ります。スマホ表示を最優先に設計するのが今の基本です。 まずトップページのデザイン案を確認し、方向性をすり合わせてから全ページに展開していきます。気になる点は遠慮なく伝えましょう。

STEP6. 原稿・写真の準備（STEP4〜5と並行）

掲載する文章（原稿）と写真を用意します。これは施主側で進めることが多く、準備の早さが全体のスピードを決めます。 文章作成が難しい場合は、ヒアリングをもとに制作側で原稿を作成することも可能です。写真も撮影代行やフリー素材で補えますが、実際の店舗・スタッフ・商品の写真があるほど信頼感が高まります。

STEP7. コーディング・実装

確定したデザインを、実際に動くWebサイトとして組み立てます（コーディング）。スマホ・タブレット・パソコンのどれで見ても崩れないように作り、表示速度や検索対策（SEO）の土台もここで整えます。専門作業なので、施主側で行うことはありません。

STEP8. 確認・修正（テスト）

完成したサイトを公開前にチェックします。文字の間違い、リンク切れ、スマホでの見え方、フォームがきちんと届くかなどを確認し、気になる箇所を修正します。 誤字や情報の最終確認は施主側にしか分からない部分なので、ここは丁寧に見ましょう。問い合わせの導線づくりは問い合わせが増えるCTA・導線改善も参考になります。

STEP9. 公開・納品

問題がなければ、サーバーにアップロードして公開します。独自ドメインの設定もこの段階で行います（ドメイン・サーバーの基礎は独自ドメインとレンタルサーバーの選び方を参照）。公開後、検索エンジンに見つけてもらうための初期設定（サイトマップ送信など）も行います。

STEP10. 運用・保守

公開はゴールではなくスタートです。情報の更新、アクセスを見ながらの改善、サーバー・ドメインの管理などを続けることで、サイトは「育つ資産」になります。自分で更新できる仕組み（CMS）があれば、お知らせやブログはご自身で発信できます（参考：小さな会社のためのコンテンツSEO入門）。

制作期間の目安はどれくらい？

「どれくらいで公開できるのか」は、ページ数と準備状況で変わります。一般的な目安は次のとおりです。

規模期間の目安主なケース
1ページ完結（LP・小規模）1〜2週間店舗・サービス紹介、キャンペーン
5〜10ページの事業者サイト1〜2か月会社・店舗の標準的なサイト
10ページ以上・機能つき2〜3か月以上撮影・原稿作成・予約機能などを含む

期間を左右する最大の要因は、原稿・写真がそろっているかと確認・返信がスムーズかです。素材が用意できていて、確認のやり取りが早ければ、上記より大きく短縮できます。

「急ぎ」にも対応できます

「開業に間に合わせたい」「今すぐ必要」というケースもあります。内容を絞り、素材がそろっていれば、短期間での公開も可能です。LE CIEL WEB DESIGN では、内容によりますが最短5営業日での公開にも対応しています。まずは希望の公開時期をお伝えください。

施主側が準備するもの【チェックリスト】

次の5つがそろっていると、相談から制作までがぐっとスムーズになります。完璧でなくても大丈夫——ある範囲で構いません。

準備するもの具体例・ポイント
① サイトの目的問い合わせを増やす／採用したい／信頼感を高める など1つに絞る
② 掲載したい内容サービス・メニュー・料金・会社/店舗情報・強み
③ 参考にしたいサイト「こんな雰囲気が好き」というサイトを2〜3つ
④ 素材ロゴ・写真・既存のパンフレットや資料
⑤ 予算・希望時期おおよその予算と、公開したい目安の時期

そろっていなくても始められます

「目的がはっきりしない」「写真がない」「文章が書けない」——どれも、ヒアリングや原稿作成・撮影代行で一緒に解決できます。準備が完璧になるまで待つ必要はありません。まずは現状のまま相談するのが、いちばんの近道です。

スムーズに進めるコツ・よくあるつまずき

制作をスムーズに、満足度高く進めるためのポイントと、ありがちなつまずきを紹介します。

コツ1. 「目的」と「ゴール」を最初に共有する

「おしゃれにしたい」だけでなく「問い合わせを月◯件に」など、目的とゴールを具体的に伝えるほど、提案の質が上がります。

コツ2. 確認・返信は早めに

制作は施主の確認を挟みながら進みます。返信が滞ると、その分だけ公開が遅れます。まとめて・早めに返すと全体が早く進みます。

コツ3. 修正の希望は具体的に伝える

「なんとなく違う」より「見出しをもっと大きく」「写真を明るく」など具体的に伝えると、少ない往復で理想に近づきます。

つまずき1. 原稿・写真の準備が止まる

もっとも多い遅延の原因がこれです。準備が難しければ、早めに「原稿作成や撮影をお願いしたい」と相談しましょう。抱え込まないのがコツです。

つまずき2. 公開後に放置してしまう

作って満足し、更新が止まるパターンです。公開はスタート。少しずつでも更新を続けることで、検索にも強くなり成果が育ちます。自作ツールとの違いを知っておくと運用の見通しも立ちます（参考：Wix・ペライチとプロ制作の違い）。

まとめ

ホームページ制作は、相談 → ヒアリング → 設計 → デザイン → 実装 → 公開 → 運用という流れで進みます。サーバー設定やコーディングといった専門作業はすべて制作側が担当するため、施主側がやることは「目的を伝える」「内容を確認する」「写真や文章を用意する」の3つだけ。 期間は素材の準備と確認のスピードしだいで大きく変わり、内容を絞れば最短5営業日での公開も可能です。
