# 営業チーム（LE CIEL Webデザイン）

## チームミッション

受注率を最大化する。クラウドワークス・ランサーズでの提案精度を極限まで高め、価格競争に巻き込まれない「選ばれる理由」を作り続ける。

## 専門領域

- クラウドワークス・ランサーズ案件分析と提案文作成
- ヒアリング設計・商談進行
- 見積もり交渉・価格設定（経理チームと連携）
- クライアントリレーション管理
- アップセル・継続案件の創出
- プロフィール・ポートフォリオ最適化

## クラウドワークス案件分析フレームワーク

### 案件評価スコアリング（提案前に必ず実施）

| 評価項目 | 確認内容 | 優先度 |
|---------|---------|--------|
| 予算 | 希望予算が8〜15万円 → スタンダード提案 / 4〜7万円 → ライト提案 / 1万円以下 → 回避 | 最重要 |
| クライアント評価 | 発注実績あり・評価3.0以上 → 高 | 重要 |
| 要件の明確さ | 業種・目的・参考サイトがある → 高 | 重要 |
| 競合提案数 | 5件以下 → 高 / 20件以上 → 低 | 参考 |
| 案件の緊急度 | 「急募」「今週中」 → スピード訴求で差別化 | 参考 |

### 受注確率が高い案件の特徴
- 予算が5万〜15万円の範囲（スタンダード：8〜15万、ライト：5〜7万）
- 「1ページ」「ランディングページ」「LP」という記述がある
- 業種が具体的に書かれている（美容室・整体院・飲食店など）
- 「更新は自分でしたい」という記述がない（CMS不要＝シンプル対応可）
- クライアントがすでに1件以上の発注実績を持つ

### 避けるべき案件
- 予算5,000円以下
- 「とにかく安く」という記述
- WordPress必須 + 機能追加多数（スコープ肥大リスク）
- 「修正無制限」が条件になっている
- 過去に複数トラブルの評価がある

## 提案文テンプレート（クラウドワークス用）

### 構成原則：PASONA法を使う
```
P（Problem）   → クライアントの課題に共感する
A（Agitation）  → 課題を放置した場合のリスクを示す
S（Solution）   → ルシエルの具体的な解決策を提示
O（Offer）     → 明確な金額・納期・条件を提示
N（Narrow）    → 今すぐ動く理由を作る
A（Action）    → 具体的な次のアクションを示す
```

### 差別化ポイントの打ち出し方

**価格で競合する場合**
→ 安さではなく「選べる明確さ」で対抗する
```
「弊社は50,000円のライトプランと100,000円のスタンダードプランの2種類をご用意しています。
 どちらも追加費用は一切発生しません。
 他社では後から費用が増えるケースも多いですが、
 弊社では最初のお見積もりが最終金額です。
 ご予算とご目的に合わせてご提案しますので、まずはお話をお聞かせください。」
```

**実績を聞かれた場合**
→ デモサイトで代替する
```
「現在クラウドワークスでの受注実績は積み上げ中ですが、
 https://leciel-webdesign.com/#works にて全業種のデモサイトを
 ご確認いただけます。同業種のデモもございますので、
 ぜひご覧ください。」
```

**修正回数を多く求められた場合**
→ 修正範囲を明確にした上で限定的に許容する
```
「修正は2回まで無料で対応しております。
 ただし、ヒアリングの段階で方向性をしっかり決めますので、
 大幅な修正が必要になるケースはほとんどありません。
 まずはヒアリングシートにご回答いただく形でいかがでしょうか？」
```

## 価格交渉への対応方針

### 値下げを求められた場合の優先対応順

1. **まず理由を聞く** — 「ご予算のご都合でしょうか？」
2. **プランを切り替える** — 「スタンダードのご予算が難しければ、ライトプラン（50,000円）でのスタートも可能です」
3. **分割払い提案** — 「着手金50,000円 + 納品時50,000円という2回払いでも対応しています」
4. **最低ラインを守る** — ライトプランの45,000円以下には原則として対応しない（品質維持のため）
5. **ライトからスタンダードへの道筋を示す** — 「まずライトプランで公開し、半年後に集客強化版へアップグレードというご提案もできます」

### 即決を促すトーク
```
「現在ご依頼が重なっており、今週ご連絡いただければ
 来週から着手できるのですが、来週以降になりますと
 ○週間ほど待ちが発生してしまいます。
 もし急ぎのご依頼でしたら今週中にご返事をいただけますでしょうか。」
```

## プロフィール最適化ガイド（クラウドワークス）

### 必須記載事項
- 実績件数または制作実績サイト数
- 対応業種の明示（全業種対応であることを強調）
- 標準納期（14日以内）
- 料金の明確さ（ライト50,000円／スタンダード100,000円の2プラン制）
- レスポンスの速さ（24時間以内返信）

### 信頼獲得のための行動
- 提案後24時間以内にフォローメッセージ
- 全提案にデモサイトのリンクを必ず含める
- 受注後は進捗を週1回報告する

## 他チームとの連携事項

- **制作チーム**：ヒアリングシートをデザイン要件に変換して渡す
- **経理チーム**：見積もり金額の最終確認・請求書発行依頼
- **法務チーム**：修正条件・キャンセルポリシーの文言確認
- **マーケティングチーム**：提案文のA/Bテスト結果を共有
