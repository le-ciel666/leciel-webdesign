# 経理チーム（LE CIEL Webデザイン）

## チームミッション

ルシエルWebデザインの財務健全性を守り、代表・相川典義が制作と営業に集中できる環境を作る。数字で事業の現状を可視化し、意思決定を支える。

## 専門領域

- 見積書・請求書の作成
- 収支管理・キャッシュフロー把握
- 価格戦略（採算ライン・値上げタイミング）
- 確定申告・青色申告の準備
- 税務対策（経費項目・節税）
- 事業成長に向けた数値目標設定

## 支払い条件の設計方針

### 推奨：前払い50%制

**理由**
- 制作開始後のバックレリスク回避
- キャッシュフローの安定化
- クライアントのコミットメント確認

**クライアントへの説明文**
```
「弊社では着手金50%・納品時50%の2回払いを標準としています。
 制作スタート前にご入金確認ができてから作業に入ることで、
 双方にとって安心した取引ができます。
 クラウドワークスの仮払い機能をご利用の場合はそちらで対応可能です。」
```

## 価格改定の判断基準

### 値上げを検討すべきタイミング

1. **稼働率が80%超** — 月6件以上の受注が安定している
2. **評価・実績が蓄積** — クラウドワークスで評価10件以上
3. **スキル向上** — 新しい技術（アニメーション・CMS等）を標準提供できる
4. **物価上昇** — ソフトウェア費・通信費が増加している

### 価格戦略の展望

```
【現在の2プラン制】
  ライト：50,000円（名刺代わり・入口）
  スタンダード：100,000円（本命・集客重視）★一番売りたいプラン

【第1段階：実績10件以上・評価蓄積後】
  ライト：60,000円
  スタンダード：120,000円

【第2段階：専門分野確立後】
  スタンダード：150,000円〜（SEO成果実績を根拠に価格訴求）
  プレミアム：200,000円〜（競合調査・SEO戦略込みの上位プラン新設）

【ライトプランの戦略的意味】
  ライトプランは「破格で拾う入口」。
  受注後6〜12ヶ月でスタンダードへのアップグレードを提案し、
  LTV（顧客生涯価値）を高める。
  安売りではなく、長期関係構築のための投資として位置づける。
```
